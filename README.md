# Frontend

Công nghệ sử dụng: React 18, React Router v6

Clone project

    git clone git@github.com:TropicalDog17/project-1.git
    cd project-1
    git checkout frontend

Cài đặt các package cần thiết:

    npm install
 
Khởi chạy ứng dụng:

    npx webpack serve
hoặc:

    npm run start
