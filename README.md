# Viết giao diện cho API

Công nghệ sử dụng: React 18, React Router v6

Clone project

    git clone git@gitlab.com:tuantran_1702/training_exercise.git
    cd training_exercise
    git checkout bt6

Cài đặt các package cần thiết:

    npm install
 
Khởi chạy ứng dụng:

    npx webpack serve
hoặc:

    npm run start
